# Task 1: Microcontroller Selection and Setup

## Objective
To configure an Arduino UNO and upload a basic LED blink program.

## Components Used
- Arduino UNO
- LED
- Jumper Wires

## Connections
- LED Anode (+) → Pin 13
- LED Cathode (-) → GND

## Program
The Arduino turns the LED ON for 1 second and OFF for 1 second repeatedly.

## Result
The LED blinked successfully, confirming proper microcontroller setup and code execution.
